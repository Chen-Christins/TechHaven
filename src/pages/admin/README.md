# 管理中心系统

这是一个独立于博客业务系统的管理中心，提供完整的后台管理功能。

## 🚀 功能特性

### 已实现功能

#### 1. 仪表盘 (`/admin`)

- **数据统计卡片**: 显示用户总数、文章总数、评论总数、今日访问量等关键指标
- **访问趋势图表**: 访问量趋势可视化（预留图表组件接口）
- **快速操作**: 提供常用功能的快速入口
- **最近活动**: 显示系统内的最新动态
- **最近用户**: 展示最新注册的用户列表

#### 2. 用户管理 (`/admin/users`)

- **用户列表**: 展示所有用户信息，包括头像、用户名、邮箱、角色、状态等
- **分页功能**: 每页显示15条数据，支持页码导航
- **高级筛选**:
  - 搜索用户（用户名/邮箱）
  - 按角色筛选（管理员/版主/普通用户）
  - 按状态筛选（活跃/非活跃/待审核）
  - 按注册时间筛选
- **用户操作**: 编辑、查看详情、删除用户
- **统计信息**: 总用户数、活跃用户、新用户、非活跃用户统计

### 系统设计

#### 响应式布局

- **桌面端**: 固定侧边栏 + 主内容区
- **移动端**: 可收缩侧边栏 + 顶部导航
- **自适应**: 支持各种屏幕尺寸

#### 主题系统

- 完全支持暗色/亮色主题切换
- 使用CSS变量实现主题一致性
- 平滑的主题切换动画

## 📱 使用说明

### 访问管理中心

1. 启动开发服务器：`npm run dev`
2. 在浏览器中访问：`http://localhost:5175/admin`
3. 或通过主导航栏的"管理"链接进入

### 用户管理操作

1. **查看用户列表**: 默认显示所有用户，每页15条
2. **搜索用户**: 在搜索框中输入用户名或邮箱
3. **筛选用户**: 使用下拉菜单按角色、状态、时间筛选
4. **分页导航**: 使用底部分页控件浏览所有用户
5. **用户操作**: 点击操作按钮进行编辑、查看或删除

### 仪表盘功能

1. **查看统计**: 顶部卡片显示关键数据指标
2. **快速操作**: 点击快速操作卡片跳转到相应功能
3. **查看动态**: 最近活动区域显示系统最新动态
4. **用户概览**: 最近用户区域显示新注册用户

## 🔧 扩展开发

### 添加新的管理页面

1. 在 `src/pages/admin/` 目录下创建新的页面组件
2. 在 `RouterConfig.tsx` 中添加路由配置
3. 在 `AdminLayout.tsx` 的导航配置中添加菜单项

### 自定义样式

- 所有样式使用CSS变量，确保主题一致性
- 响应式断点：768px（移动端）
- 动画效果统一使用0.2-0.3s缓动

### 数据集成

- 当前使用模拟数据，可替换为真实API调用
- 数据接口定义在组件顶部的interface中
- 支持异步加载和错误处理
